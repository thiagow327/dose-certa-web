# Dose Certa Web

Interface web para controle de medicamentos de idosos. Permite cadastrar remédios com dosagens e frequências, registrar doses tomadas e acompanhar o histórico de aderência ao tratamento, tudo em uma única tela.

Desenvolvida com **HTML**, **CSS**, **JavaScript** puro e **Bootstrap 5**.

---

## Requisitos

- Um navegador web (Chrome, Firefox, Edge, etc.)
- A [Dose Certa API](https://github.com/thiagow327/dose-certa-api) rodando em `http://localhost:5000`

---

## Instalação

### 1. Clone o repositório

```bash
git clone https://github.com/thiagow327/dose-certa-web.git
cd dose-certa-web
```

### 2. Inicie a API

A interface consome a [Dose Certa API](https://github.com/thiagow327/dose-certa-api). Certifique-se de que ela esteja rodando em `http://localhost:5000` antes de abrir a aplicação.

---

## Inicialização

Abra o arquivo `index.html` diretamente no navegador.

> No VS Code, também é possível usar a extensão **Live Server**.
